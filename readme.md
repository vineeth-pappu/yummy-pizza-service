  - Order in USD or EURO
  - Easy Checkout

### Tech

It is build Laravel and Mysql:
* Laravel
* Mysql


### Installation

Yummy Pizza requires PHP v7.0+ to run.

Install the dependencies and devDependencies and start the server.

```sh
$ composer update
$ php artisan migrate
$ php artisan serve
```
